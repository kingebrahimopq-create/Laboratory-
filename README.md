نظام **MY LAB** هو نظام متكامل ومحمي لإدارة معلومات المختبرات الطبية والسجلات الإدارية عن بعد. تم بناء وتطوير المنصة بأعلى معايير الحماية والتكامل التلقائي مع خوادم البناء لبناء تطبيقات أصلية مدمجة لهاتفك الجوال بنظام **الاندرويد (Android)** وكذا لحاسوبك بنظام **الويندوز (Windows Desktop)**.

---

## 📥 روابط تحميل التطبيقات المباشرة (Direct App Downloads)

لتسهيل التحميل ومتابعة أحدث البنايات التلقائية المصممة خصيصاً للمعمل، يمكنك استخدام الروابط التالية:

### 🤖 تطبيق الأندرويد للهواتف الذكية (Android App - APK)
* **رابط التحميل التلقائي:** يمكنك الدخول إلى [🤖 صفحة عمليات البناء (Actions)](https://github.com/kingebrahimopq-create/Laboratory-/actions) وتحميل ملف الـ APK المرفق بآخر بناء تحت اسم **MyLab-Android-App**.
* **قائمة الإصدارات الثابتة:** يمكنك الحصول عليها من [🚀 صفحة الإصدارات (Releases)](https://github.com/kingebrahimopq-create/Laboratory-/releases).

### 💻 تطبيق الويندوز لسطح المكتب (Windows App - Installer)
* **رابط التحميل التلقائي:** يمكنك الدخول إلى [💻 صفحة عمليات البناء (Actions)](https://github.com/kingebrahimopq-create/Laboratory-/actions) وتحميل ملف التثبيت المرفق بآخر بناء تحت اسم **MyLab-Windows-App**.
* **قائمة الإصدارات الثابتة:** يمكنك الحصول عليها من [🚀 صفحة الإصدارات (Releases)](https://github.com/kingebrahimopq-create/Laboratory-/releases).

> 💡 **نصيحة سريعة:** لتنزيل الملفات من صفحة **Actions**، اضغط على أحدث بناء ناجح (أخضر)، ثم انزل لأسفل الصفحة حتى تجد قسم **Artifacts** وبداخله الملفات مباشرة للتحميل بضغطة واحدة.

## ✨ المميزات الأساسية للمنصة الطبية (Core LIMS Features)

1. **🔐 حماية هوية المالك الحساسة:** جميع الحسابات والمعلومات تمت تصفيتها وتأمينها لتعتمد كلياً على رمز أمان المالك وعنوانه الإلكتروني الذي يتم تحديده بشكل حيوي وديناميكي من إعدادات الطبيب المدير في لوحة الإدارة لمنع أي استغلال أو تتبع.
2. **🧬 بوابة المصادقة بترميز الاستجابة السريعة (Public QR Verification):** التحقق التلقائي والفوري مع النتائج الطبية بالبث السحابي للتأكد المستمر من نتائج التحاليل السليمة والحد من التزييف.
3. **📁 إدارة الملفات المرضية المتكاملة:** تسجيل المريض، الملاحظات الطبية الحيوية، إدارة الفواتير والأسعار وسجلات الدفع في ثوانٍ.
4. **🔬 منصة فني الأجهزة المخبرية:** تعيين المهام ورفع النتائج، ومطابقتها للمعدلات المرجعية بدقة متطورة.
5. **🖨️ طباعة التقارير بنسق PDF فوري:** تصميم تقارير مخبرية غاية في الأناقة وبسيطة التحول لتصميم ورقي جذاب ومتناسق مع طابعات العيادة.
6. **🎯 التأمين والتحقق البيومتري (Biometric Assurance Platform):** لوحة تفعيل محاكاة بصمة الطبيب المالك لتوقيع واعتماد النتيجة السريرية وصيغ التشخيص.

## 🛠️ كيفية تشغيل المنصة محلياً للمطورين (Local Development Setup)

إذا كنت ترغب في تشغيل واجهة النظام أو تطويرها بنظام Node:

```bash
# 1. استنساخ المستودع
git clone https://github.com/kingebrahimopq-create/Laboratory-.git
cd Laboratory-

# 2. تثبيت الحزم المطلوبة متوافقة مع الأنظمة الكبيرة
npm install --legacy-peer-deps

# 3. تشغيل وضع التطوير المستمر
npm run dev
```

## 🚀 البناء التلقائي المتكامل (CI/CD Workflows)

المستودع مجهز بثلاثة خطوط بناء تلقائية فائقة السرعة معتمدة على **GitHub Actions**:
1. **أندرويد:** ترقية Java JDK إلى الإصدار 21 و Node.js 22 لضمان توافقية بناء Cordova & Capacitor بشكل كامل وخلوه من الأخطاء التجميعية.
2. **الويندوز:** بناء وتصدير حزمة Electron إلى صيغة تثبيت ممتازة (.exe) في ثوانٍ معدودة.
