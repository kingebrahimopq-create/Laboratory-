## Reporting a Vulnerability

If you discover a security vulnerability in My Lab LIMS, please follow these steps:

1. **Do not** open a public issue
2. **Do not** disclose the vulnerability publicly
3. **Do** send an email to: security@my-lab-lims.com (replace with actual email)
4. **Do** include the following information:
   - Vulnerability description
   - Steps to reproduce
   - Impact assessment
   - Suggested fix (if any)

Our security team will:
- Acknowledge your report within 48 hours
- Investigate the issue and determine severity
- Develop and test a fix
- Release the fix in the next version
- Credit you (if you wish) in the release notes

## Security Best Practices

### For Users
- Always use the latest version
- Keep your dependencies updated
- Use strong passwords
- Enable 2FA when available
- Regularly backup your data
- Review user permissions

### For Developers
- Follow OWASP Top 10 guidelines
- Sanitize all user inputs
- Use parameterized queries
